# calum-owen.github.io

This repository contains a Jekyll-based personal site scaffold created by GitHub Copilot. Replace placeholder content and add your images to `assets/images/`.

What I created:
- Basic Jekyll config and layouts
- Pages: Home, About, Recipes (collection), Portfolio, Triathlon, Contact
- Client-side search scaffold (search.json + site.js)
- Utterances comments configured for this repo
- Dinner sign-up form that opens the user's email client (recipient email is obfuscated client-side)

Next steps for you:
- Add a short bio in about.md
- Upload headshot and triathlon images to `assets/images/` and update pages
- Add recipe files to `_recipes/` or edit the placeholder
- Add project entries to portfolio.md or add project pages
- If you want emails sent server-side, set up Formspree or Netlify Forms and I can wire it up
